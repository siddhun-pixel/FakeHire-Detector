
# FakeHire Detector

An AI-based system to detect fake job and internship postings for students using NLP and Machine Learning.

## Features
- Scam keyword detection
- Email & salary analysis
- ML-based classification
- Explainable output

## Tech Stack
Python, Streamlit, Scikit-learn, NLP

## Run Locally
pip install -r requirements.txt  
streamlit run main.py
